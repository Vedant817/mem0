Get started with Mem0's Node.js SDK in under 5 minutes. This guide shows you how to install Mem0 and store your first memory.

## Prerequisites

- Node.js 14 or higher
- OpenAI API key ([Get one here](https://platform.openai.com/api-keys))

## Installation

<Steps>
<Step title="Install via npm">
```bash
npm install mem0ai
```
</Step>

<Step title="Set your API key">
```typescript
import { Memory } from 'mem0ai/oss';

const memory = new Memory();
```
</Step>

<Step title="Add your first memory">
```typescript
const messages = [
    {"role": "user", "content": "Hi, I'm Alex. I love TypeScript and gaming."},
    {"role": "assistant", "content": "Hey Alex! I'll remember your interests."}
];

const result = await memory.add(messages, { userId: "alex" });
console.log(result);
```

**Output:**
```json
{
    "results": [
        {
            "id": "mem_123abc",
            "memory": "Name is Alex. Enjoys TypeScript and gaming.",
            "event": "ADD"
        }
    ]
}
```
</Step>
</Steps>
